# allium58-zmk-config

Keymaps can be updated manually or through a [web interface](https://nickcoutsos.github.io/keymap-editor/).

<img src="keymap-drawer/lily58.svg"  alt="my_keymap">

Keymap image generated by [caksoylar/keymap-drawer](https://github.com/caksoylar/keymap-drawer).

Changes to the keymap made through Pull Requests will have commits
automatically added to update the related keymap-drawer keymaps and SVGs.

To manually update the image to match the current keymap:

```
uv tool install keymap-drawer
keymap parse -z config/lily58.keymap > keymap-drawer/lily58.yml
keymap -c keymap-drawer/config.yml draw keymap.yml -o keymap-drawer/lily58.svg
```
