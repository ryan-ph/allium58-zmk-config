# allium58-zmk-config

Keymaps can be updated manually or through a [web interface](https://nickcoutsos.github.io/keymap-editor/).

<img src="keymap-drawer/keymap.svg"  alt="my_keymap">

Keymap image generated by [caksoylar/keymap-drawer](https://github.com/caksoylar/keymap-drawer).
To update the image to match the current keymap:

```
uv tool install keymap-drawer
keymap parse -z config/lily58.keymap > keymap-drawer/keymap.yml
keymap -c keymap-drawer/config.yml draw keymap.yml -o keymap-drawer/keymap.svg
```
